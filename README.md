# Практика

Электронное портфолио по результатам прохождения учебной (эксплуатационной) практики.

## Состав репозитория

- `задание_на_практику_1_курс_Фамилия_И.О.pdf` - документ «Задание».
- `отчет_по_практике_1_курс_Фамилия_И.О.pdf` - итоговый отчет по практике.
- `Задание ИСР 1.1.pdf` - GitHub / GitLab.
- `Задание ИСР 1.2.pdf` - аннотированный список источников по философским проблемам информатики.
- `Задание ИСР 1.3.pdf` - история информатики, интеллект-карта.
- `Задание ИСР 1.4.pdf` - стандарты и спецификации в сфере ИТ.
- `Задание ИСР 1.5.pdf` - комплекс физических упражнений для программиста.
- `Задание ИСР 1.6.pdf` - инструкция по охране труда программиста.
- `Задание ИСР 1.7.pdf` - квалификационный справочник.
- `Задание ИСР 1.8.pdf` - автоматизированная инсталляция ПО.
- `Задание ИСР 1.9.pdf` - АРМ специалиста, интеллект-карта.
- `Задание ВСР 2.1.pdf` - описание рабочего места программиста, интеллект-карта.
- `Задание ВСР 2.2.pdf` - первая помощь при электротравме.
- `Задание ВСР 2.3.pdf` - системное ПО информационно-вычислительной системы.
- `Задание ВСР 2.4.pdf` - технические средства информационно-вычислительной системы.
- `Задание ВСР 2.5.pdf` - искусственный интеллект: генерация текста.

## Что нужно заменить после загрузки

В файлах `задание_на_практику_1_курс_Фамилия_И.О.pdf` и `отчет_по_практике_1_курс_Фамилия_И.О.pdf` используются шаблонные ФИО и ссылка на репозиторий. При необходимости нужно открыть DOCX-исходник, заменить ФИО и ссылку на фактические данные, затем сохранить новый PDF.
